PHP 后端管理平台

基于 RBAC 权限模型的后端管理系统

技术栈

  • 后端框架: 原生 PHP (无框架)
  • 数据库: MySQL
  • 前端: HTML5, CSS3, JavaScript, Bootstrap 5
  • 认证: 基于会话的认证
  • 权限管理: RBAC (角色基于访问控制)

项目结构

admin-panel/
├── config/          # 配置文件
├── controllers/     # 控制器
├── models/          # 模型
├── views/           # 视图
│   ├── auth/        # 认证相关视图
│   ├── users/       # 用户管理视图
│   ├── roles/       # 角色管理视图
│   ├── permissions/ # 权限管理视图
│   ├── data/        # 数据管理视图
│   └── layouts/     # 布局文件
├── public/          # 公共资源
│   ├── css/         # CSS 文件
│   └── js/          # JavaScript 文件
├── includes/        # 包含文件
├── middleware/      # 中间件
├── helpers/         # 辅助函数
└── database/        # 数据库脚本

安装说明

  1. 创建数据库
    CREATE DATABASE admin_panel CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
  2. 导入数据库脚本

    使用 MySQL 客户端导入 database/init.sql 文件

  3. 配置数据库连接

    编辑 config/config.php 文件，修改数据库连接信息：

    define('DB_HOST', 'localhost');
define('DB_NAME', 'admin_panel');
define('DB_USER', 'root');
define('DB_PASS', '');
  4. 部署到 Web 服务器

    将项目部署到 Apache 或 Nginx 服务器，确保 PHP 版本 >= 7.4

  5. 访问管理后台

    打开浏览器访问：http://your-domain/admin-panel/login.php

使用指南

1. 登录系统

使用默认账号登录系统后，您将看到管理后台仪表盘。

2. 管理角色和权限

在左侧菜单栏中，您可以：

  • 点击 角色管理 创建和管理角色
  • 点击 权限管理 创建和管理权限
  • 为角色分配相应的权限

3. 管理用户

点击 用户管理，您可以：

  • 创建新用户
  • 编辑现有用户信息
  • 为用户分配角色
  • 删除用户

4. 管理数据

点击 数据管理，您可以：

  • 查看产品列表
  • 创建、编辑、删除产品
  • 按名称、分类、状态搜索产品

安全建议

  • 修改默认管理员密码
  • 定期备份数据库
  • 使用 HTTPS 协议
  • 限制登录尝试次数
  • 定期更新 PHP 版本
