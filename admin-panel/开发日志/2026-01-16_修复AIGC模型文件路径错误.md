# 2026-01-16 修复AIGC模型文件路径错误

## 问题概述

修复了AIGC模型文件中出现的路径错误：

```
Warning: require_once(/www/wwwroot/cz.younger-car.com/config/config.php): Failed to open stream: No such file or directory in /www/wwwroot/cz.younger-car.com/admin-panel/models/AIGC.php on line 2
```

## 修复内容

**修改文件**：
- `admin-panel/models/AIGC.php`

**修复方案**：
修正了配置文件和其他辅助文件的路径：

```php
// 原代码
require_once APP_ROOT . '/config/config.php';
require_once APP_ROOT . '/includes/database.php';
require_once APP_ROOT . '/includes/Logger.php';

// 修复后的代码
require_once APP_ROOT . '/admin-panel/config/config.php';
require_once APP_ROOT . '/admin-panel/includes/database.php';
require_once APP_ROOT . '/admin-panel/includes/Logger.php';
```

## 验证结果

修复后，AIGC模块能够正常加载配置文件和数据库连接，不再出现文件路径错误。

## 影响范围

此修复仅影响AIGC模块的功能，不会对系统其他模块产生影响。系统现在可以正常处理AI图片处理任务。
