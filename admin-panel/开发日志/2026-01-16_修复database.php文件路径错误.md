# 2026-01-16 修复database.php文件路径错误

## 问题概述

修复了database.php文件中出现的路径错误：

```
Warning: require_once(/www/wwwroot/cz.younger-car.com/config/config.php): Failed to open stream: No such file or directory in /www/wwwroot/cz.younger-car.com/admin-panel/includes/database.php on line 2
```

## 修复内容

**修改文件**：
- `admin-panel/includes/database.php`

**修复方案**：
修正了配置文件的路径：

```php
// 原代码
require_once APP_ROOT . '/config/config.php';

// 修复后的代码
require_once APP_ROOT . '/admin-panel/config/config.php';
```

## 验证结果

修复后，database.php文件能够正常加载配置文件，不再出现路径错误。

## 影响范围

此修复影响所有使用Database类的模块，包括AIGC模块、用户管理、商品管理等。系统现在可以正常建立数据库连接，处理各种数据操作。
